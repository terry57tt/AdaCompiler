# Start the project

## Linux & Mac
```shell
# in root project PCL

# to setup the gradle project 
./gradlew build

# all tests
./gradlew test

# specific test 
./gradlew test --tests ProjectConfigurationTest

# help menu 
./gradlew tasks
```
